OUR PAGE — Cinematic Proposal Webpage
======================================

FOLDER STRUCTURE:
  index.html       ← Open this in browser
  music/
    music1.mp3     ← Track 1 (Pages 1-4)
    music2.mp3     ← Track 2 (Pages 5-10)
    music3.mp3     ← Track 3 (Page 11-12)

HOW TO USE:
  1. Keep all files together — do NOT move index.html out of the folder
  2. Open index.html in Chrome (mobile or desktop)
  3. Tap screen to begin and start music

NAVIGATION:
  Tap    → advance dialogues / start music
  Swipe  → move between pages (unlocks after dialogue ends)
  Ring   → touch it on Page 11 to trigger celebration
  Page 12 → tap symbols to remove them, find hidden YES button

NOTE: Must be opened from the folder (not dragged alone).
      Works offline — no internet needed (except Google Fonts for typography).
